Place the bundled `aria2c` binary here.

Expected paths:

- `resources/aria2/win32-x64/aria2c.exe`
- `resources/aria2/win32-arm64/aria2c.exe`
- `resources/aria2/linux-x64/aria2c`
- `resources/aria2/linux-arm64/aria2c`
- `resources/aria2/darwin-x64/aria2c`
- `resources/aria2/darwin-arm64/aria2c`

For local development, `ARIA2C_BIN` can override the binary path.
